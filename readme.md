# Web streaming RTC
    WebRTC api

## This project use
    * Express, SocketIO, PeerJS

## Signal
    * Read more [Peer](https://peerjs.com/)

## How to run
    **Run server**
    1. clone app.
    2. yarn or npm install.
    3. yarn start or npm run start.

    **Run server signal**
    1. install -g peer
    2. peerjs --port 3003 || (any port)
